Getting Started with RayEditor

Installation

Install via npm: npm install @rohanyeole/ray-editor

This is a bold word and an italic phrase together.

Features

  • Zero dependencies
  • TypeScript support
  • Slash commands — type / to open palette
  1. Install the package
  2. Import the CSS
  3. Create an instance
const editor = new RayEditor('container', {
  toolbar: ['bold', 'italic', 'link'],
  theme: 'dark',
});
editor.on('content:change', ({ html }) => {
  console.log(html);
});
