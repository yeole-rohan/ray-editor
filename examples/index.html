Getting started

    
Up and running in 60 seconds

    
Choose npm for bundled apps or CDN for quick prototypes.


    

      

        
        
        
      


      

        

          

            

              $
              npm install @rohanyeole/ray-editor
              
            

          

          

            
main.js

            
import { RayEditor } from '@rohanyeole/ray-editor';
import '@rohanyeole/ray-editor/css';

const editor = new RayEditor('editor', {
  theme: 'light',
  markdownShortcuts: true,
  slashCommands: true,
  wordCount: true,
  onChange: (html) => console.log(html),
});

API Reference

    
Clean, predictable API

    
Everything you need to get, set, and extend the editor.


    

      

        
Core API

        
const editor = new RayEditor('elementId', options);

// Content
editor.getContent();            // → clean HTML string
editor.setContent(html);       // set HTML programmatically

// Theme
editor.setTheme('dark');        // 'light' | 'dark' | 'auto'

// Read-only
editor.setReadOnly(true);      // lock/unlock editing

// Word count
editor.getWordCount();         // → { words: number, chars: number }

// Events
editor.on('content:change', (html) => {});
editor.off('content:change', handler);

// Plugins
editor.use({ name: 'my-plugin', install(ed) { /* ... */ } });

// Slash commands
editor.registerSlashCommand({
  name: "Today's Date", icon: '📅',
  description: 'Insert today\'s date',
  action: () => document.execCommand('insertText', false, new Date().toLocaleDateString()),
});

// Cleanup
editor.destroy();
